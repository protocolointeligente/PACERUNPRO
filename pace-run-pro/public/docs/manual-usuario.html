1. Introdução

Geral

Bem-vindo ao PACE RUN PRO — a plataforma completa para assessorias esportivas de corrida, treinadores e atletas. Este manual foi escrito para ajudá-lo a aproveitar ao máximo todas as funcionalidades do sistema, independente do seu nível de experiência com tecnologia.

O PACE RUN PRO é uma aplicação web acessada pelo navegador de qualquer dispositivo — computador, tablet ou celular. Não é necessário instalar nada. Todos os seus dados ficam salvos automaticamente na nuvem.

Três tipos de usuário

O sistema foi construído para três perfis distintos, com funcionalidades específicas para cada um:

🏢
Assessoria
Gerencia tudo
gerencia
👨‍💼
Treinadores
Prescrevem treinos
treinam
🏃
Atletas
Executam treinos
🏢 Assessoria (Admin)
Gerencia treinadores, configurações, financeiro e marca da plataforma
👨‍💼 Treinador (Coach)
Cria e prescreve treinos, monitora atletas, analisa desempenho
🏃 Atleta (Aluno)
Visualiza e executa treinos, acessa ferramentas de análise e IA
Acesso
Qualquer navegador web — Chrome, Safari, Firefox, Edge
💡 Dica

Ao fazer login, o sistema identifica automaticamente qual perfil você possui e exibe o painel correto. Não é necessário selecionar manualmente.

2. Primeiros Passos

Todos os perfis
2.1

Como fazer login

https:// www. pacerunpro.com.br
  1. 1 Acesse www.pacerunpro.com.br em qualquer navegador
  2. 2 Clique no botão "Entrar na sua conta" na página inicial
  3. 3 Escolha como entrar:
    Opção A — Google: clique em "Continuar com Google" e selecione sua conta. Mais rápido e seguro.
    Opção B — E-mail: informe seu e-mail e senha cadastrados, depois clique em "Entrar"
  4. 4 O sistema detecta seu perfil automaticamente e redireciona para o painel correto
⚠️ Atenção

Se você tiver uma conta Google e também uma conta com e-mail/senha usando o mesmo endereço de e-mail, use sempre o mesmo método de login. Misturar os dois pode criar contas separadas.

2.2

Navegação geral

  • No computador: menu lateral esquerdo com todas as seções disponíveis para seu perfil
  • No celular: ícones na barra de navegação inferior — toque para navegar entre seções
  • Clique no ⚡ logo (canto superior esquerdo) para voltar ao início a qualquer momento
  • Clique no ícone de usuário (canto superior direito) para acessar seu perfil e a opção de sair
  • O menu destaca em roxo a seção onde você está atualmente
2.3

Como adicionar à tela do celular

O PACE RUN PRO funciona como um aplicativo quando adicionado à tela inicial. Funciona mesmo com conexão instável.

📱 iPhone (Safari)
  1. 1Abra o site no Safari (não funciona no Chrome do iPhone)
  2. 2Toque no ícone de compartilhar (quadrado com seta para cima) na barra inferior
  3. 3Role a lista e toque em "Adicionar à Tela de Início"
  4. 4Confirme o nome e toque em "Adicionar"
🤖 Android (Chrome)
  1. 1Abra o site no Chrome
  2. 2Toque nos três pontos (menu) no canto superior direito
  3. 3Toque em "Adicionar à tela inicial"
  4. 4Toque em "Adicionar" para confirmar
💡 Dica

Ao "instalar" desta forma, o PACE RUN PRO ganha um ícone na sua tela e abre sem a barra de endereços do navegador, parecendo um app nativo. Seus dados continuam no servidor — se remover o atalho, basta adicionar novamente.

3. Perfil Assessoria

🏢 Assessoria

O perfil Assessoria é o nível mais alto do sistema. O administrador tem acesso completo a todas as configurações, gestão de treinadores e controle financeiro.

3.1

Dashboard Administrativo

https://app / admin/dashboard

Ao entrar como administrador, você vê o painel central com uma visão completa do negócio:

Total de atletas
Número de atletas ativos na plataforma
Treinadores ativos
Quantos treinadores estão com acesso habilitado
MRR
Receita Recorrente Mensal — total de assinaturas ativas
Crescimento mensal
Variação percentual de atletas em relação ao mês anterior
💡 Dica

Os cards com bordas vermelhas no dashboard indicam alertas que precisam de atenção imediata, como pagamentos vencidos ou treinadores sem atletas há mais de 30 dias.

3.2

Gerenciar Treinadores

https://app / admin/assessorias
Como adicionar um treinador
  1. 1Clique no botão "+ Novo treinador" no canto superior direito
  2. 2Preencha o nome completo e o e-mail profissional do treinador
  3. 3Clique em "Enviar convite" — o sistema envia um e-mail de boas-vindas automaticamente
  4. 4O treinador segue o link no e-mail para definir sua senha e ativar a conta
Permissões e acesso
  • Cada treinador acessa apenas os atletas vinculados a ele — não vê atletas de outros treinadores
  • O administrador pode ver e acessar os atletas de todos os treinadores
  • Para remover acesso: clique no nome do treinador → botão "Desativar" — o treinador perde o acesso imediatamente, mas os dados são preservados

Configuração White-label Novo

🌐 Domínio personalizado

💡 Dica

Após clicar em "Salvar", as mudanças de nome e cor aparecem em até 1 minuto para todos os usuários. Recarregue a página se não aparecer.

3.4

Financeiro e Planos

https://app / admin/financeiro
  • Visualize cobranças Pagas, Pendentes e Vencidas
  • Gere relatório mensal completo — clique em "Exportar" para baixar em CSV ou PDF
  • Integração nativa com Asaas para cobranças via Pix, boleto e cartão de crédito
  • Histórico de todas as transações com filtro por período e status
⚠️ Atenção

Cobranças vencidas há mais de 7 dias suspendem automaticamente o acesso do atleta. O acesso é restaurado assim que o pagamento é confirmado.

4. Perfil Treinador

👨‍💼 Treinador

O treinador é o coração operacional da plataforma. Aqui você gerencia seus atletas, prescreve treinos, monitora desempenho e acompanha o progresso de cada aluno em tempo real.

4.1

Dashboard do Treinador

https://app / treinador/dashboard

Ao entrar, você vê um resumo completo da sua equipe de atletas:

Atletas ativos
Total de atletas sob sua orientação
Treinos esta semana
Quantos treinos foram prescritos e quantos já foram completados
Alertas pendentes
Situações que precisam da sua atenção hoje
Taxa de aderência
Percentual de treinos completados no período

Os cards de atletas usam cores para indicar urgência:

  • ● Vermelho — requer atenção imediata (dor reportada, sem treinar há 7+ dias)
  • ● Amarelo — observar (aderência baixa, volume abaixo da meta)
  • ● Verde — tudo certo, atleta evoluindo normalmente
💡 Dica

Clique em qualquer card de atleta para abrir o perfil completo com histórico, métricas e todos os treinos prescritos.

4.2

Gerenciar Atletas

https://app / treinador/alunos
Ver a lista de atletas

A tabela mostra: foto, nome, plano atual, data da última atividade e status. Use os filtros no topo para encontrar atletas por status, plano ou nível.

Ver perfil individual

Clique no nome do atleta para abrir o perfil completo: histórico de treinos, métricas de desempenho, evolução de pace, FC, volume semanal e todos os treinos prescritos.

Adicionar novo atleta
  1. 1Clique em "+ Novo atleta" no canto superior direito
  2. 2Preencha nome e e-mail, ou compartilhe o link /cadastro para o atleta preencher os próprios dados
  3. 3O atleta recebe um e-mail de boas-vindas com instruções para criar a senha
4.3

Central de Alertas

https://app / treinador/alertas

O sistema gera alertas automáticos com base no comportamento dos atletas. Você não precisa verificar cada atleta individualmente — os alertas chegam até você.

Tipos de alerta
  • 🔴 Alto — atleta sem treinar há 7+ dias, dor reportada na sessão, frequência cardíaca muito acima do esperado
  • 🟡 Médio — aderência abaixo de 75% na semana, volume total abaixo da meta do plano
  • 🔵 Info — melhora significativa de performance, treino executado de forma excelente, novo recorde pessoal
Gerenciar alertas
  • Filtrar por severidade: clique nos botões Todos / Alto / Médio / Info no topo
  • Marcar como lido: clique em "Marcar como lido" no card do alerta — ele é arquivado
  • Marcar todos de uma vez: botão "Marcar todos como lidos" no canto superior direito
4.4

Análise Semanal Automática

https://app / treinador/analise-semanal

Todo domingo, o sistema gera automaticamente um relatório consolidado de todos os seus atletas com os dados da semana encerrada.

  • Navegar entre semanas: use as setas ← e → no topo para ver semanas anteriores
  • Filtros de risco: botões "Alto risco" e "Médio risco" para focar apenas nos atletas que precisam de atenção
  • Card por atleta: volume total, aderência (%), pace médio, carga de treino acumulada
  • Variação semanal: ▲ indica aumento, ▼ indica queda em relação à semana anterior
  • Recomendação automática: cada card traz uma sugestão do que fazer com aquele atleta na próxima semana
💡 Dica

Use a análise semanal como ponto de partida para ajustar os treinos da semana seguinte. Clique no nome do atleta direto do card para abrir o perfil e fazer a prescrição.

4.5

Prescrição de Corrida

https://app / treinador/prescricao/corrida
  1. 1Selecione o atleta no dropdown — comece digitando o nome para filtrar
  2. 2Tipo de treino: escolha entre Fácil, Moderado, Forte, Longão, Regenerativo ou Específico
  3. 3Distância: informe a distância em km (ex: 10)
  4. 4Pace alvo: informe o pace em min/km (ex: 5:30) — use o formato correto com dois pontos
  5. 5Data: clique no calendário e selecione quando o treino deve ser realizado
  6. 6Observações: campo livre — escreva instruções específicas, aquecimento, sprints etc.
  7. 7Clique em "Salvar treino" — o treino aparece instantaneamente no calendário do atleta
💡 Dica

Você pode prescrever treinos com semanas de antecedência. O atleta vê todos os treinos agendados no calendário e recebe notificação no dia anterior ao treino.

4.6

Prescrição de Força

https://app / treinador/prescricao/forca
Criar uma sessão de força
  1. 1Clique em "+ Nova sessão"
  2. 2Dê um nome à sessão (ex: "Força Membros Inferiores A") e defina a data
  3. 3Selecione o objetivo: Funcional, Hipertrofia ou Potência
  4. 4Clique em "Adicionar exercício" e busque na biblioteca (mais de 30 exercícios com GIFs)
  5. 5Para cada exercício, defina: séries × repetições × tempo de descanso (em segundos)
  6. 6Reordenar: arraste e solte os exercícios para mudar a ordem da sessão
  7. 7Clique em "Salvar sessão" — o atleta vê a sessão completa com imagens e timer integrado
💡 Dica

A biblioteca de exercícios tem GIFs demonstrativos. O atleta vê exatamente como executar cada movimento, reduzindo erros técnicos sem precisar de supervisão presencial.

4.7

Periodização Novo

https://app / treinador/prescricao/periodizacao

Monte um plano de treinamento estruturado para semanas ou meses, com progressão automática de carga.

  1. 1Criar macrociclo: clique em "+ Novo macrociclo" → dê um nome (ex: "Preparatório São Silvestre") → defina data de início e fim
  2. 2Fases automáticas: o sistema sugere a divisão Base → Construção → Específico → Polimento → Recuperação com proporções recomendadas
  3. 3Ajuste manual: arraste as barras das fases no calendário para aumentar ou reduzir a duração de cada uma
  4. 4Volume por fase: o sistema calcula progressão automática de carga — você pode editar os valores manualmente
  5. 5Vincular: aplique o plano para um atleta específico ou para um grupo inteiro de atletas de uma só vez
4.8

Relatórios

https://app / treinador/relatorios
  1. 1Selecione o atleta no dropdown
  2. 2Escolha o período usando o seletor de datas
  3. 3Selecione o tipo de relatório: Semanal, Mensal ou Por Prova
  4. 4Clique em "Gerar relatório" — uma janela de pré-visualização se abre
  5. 5Salvar como PDF: na janela de impressão, selecione "Salvar como PDF" no destino de impressão
  6. 6Exportar CSV: use o botão separado "Exportar CSV" para obter todos os dados em planilha
4.9

CRM de Leads Novo

https://app / treinador/crm

Gerencie potenciais novos alunos com um pipeline visual no estilo Kanban.

Pipeline Kanban

Os leads são organizados em colunas que representam o estágio da negociação:

Novo → Contato → Proposta → Ganho
Adicionar um lead
  1. 1Clique em "+ Novo lead"
  2. 2Preencha: nome, e-mail, telefone, origem (como te encontrou) e valor estimado do plano
  3. 3Clique em "Salvar" — o lead aparece na coluna "Novo"
  4. 4Para avançar o estágio: clique em "Mover →" no card do lead
  • Alternar para lista: botão "Lista" no topo para ver todos os leads em formato de tabela
  • Taxa de conversão: calculada automaticamente — quantos leads viraram atletas
  • Receita potencial: soma dos valores estimados de todos os leads nas colunas ativas

Calendário de Treinos

https://app / aluno/calendario

Visualização mensal com todos os seus treinos marcados por cor:

  • 🟣 Roxo — treino de corrida

  • 🟢 Verde — treino recuperativo ou caminhada

5.3

Treino de Força com Timer

https://app / aluno/forca

Execute sua sessão de força diretamente pelo celular, com timer automático e demonstrações em GIF.

  1. 1Na página de força, selecione a sessão do dia e clique em "Iniciar treino"
  2. 2O primeiro exercício é exibido com nome, GIF demonstrativo, séries e repetições
  3. 3Execute a série — o timer conta o tempo de esforço na tela
  4. 4Ao completar a série, clique em "Concluir série" — o timer de descanso inicia automaticamente com anel animado em contagem regressiva
  5. 5Um beep sonoro avisa quando o descanso termina — prepare-se para a próxima série
  6. 6Ao concluir todas as séries do exercício, clique em "Próximo exercício"
  7. 7Ao terminar todos os exercícios, aparece a tela de conclusão com o resumo da sessão
💡 Dica

Mantenha o celular com a tela ativa durante o treino. Você pode reduzir o brilho para economizar bateria, mas não feche o navegador ou o timer será interrompido.

5.4

IA Treinadora Novo

https://app / aluno/ia-treinadora

Um assistente inteligente especializado em corrida e treinamento, disponível 24 horas por dia para tirar suas dúvidas.

Como usar
  • Clique em uma das perguntas sugeridas para começar rapidamente
  • Ou escreva sua própria pergunta na caixa de texto
  • Pressione Enter ou clique no botão de envio (ícone de seta) para enviar
  • Para pular linha sem enviar, use Shift+Enter
  • A IA já conhece seu plano atual, provas agendadas e histórico recente automaticamente
Exemplos de perguntas
  • "Meu joelho está doendo depois dos treinos longos, o que faço?"
  • "Como devo me alimentar nos 3 dias antes da maratona?"
  • "Estou me sentindo muito cansado — será que estou overtreinando?"
  • "Qual pace devo usar no meu próximo longão de 25km?"
  • "Posso substituir o treino de amanhã por hoje?"
⚠️ Atenção

A IA Treinadora é um apoio complementar, não substitui seu treinador humano. Para questões de saúde, lesões ou dores persistentes, sempre consulte um médico ou fisioterapeuta.

💡 Dica

O histórico de conversa fica salvo durante a sessão ativa. Ao fechar e reabrir o aplicativo, uma nova conversa começa do zero.

5.5

Previsão de Prova

https://app / aluno/previsao

Descubra qual é o seu tempo previsto para qualquer distância com base no seu desempenho atual.

  1. 1Informe uma distância que você correu recentemente (ex: 10 km)
  2. 2Informe seu tempo naquela prova ou treino (ex: 52:00)
  3. 3O sistema usa a fórmula científica de Riegel para calcular previsões para 5k, 10k, 21k, 42k e 100k
O que você recebe
  • Tabela de previsões: tempo estimado para cada distância padrão
  • Pace médio esperado para cada prova
  • Tabela de splits: pace esperado para cada km da prova alvo
  • Estratégia de pace negativo: recomendação de começar mais devagar e acelerar nos km finais
💡 Dica

Para previsões mais precisas, use um tempo de uma prova recente (últimos 3 meses) em condições normais. Tempos em dias muito quentes ou em terrenos acidentados podem dar previsões menos precisas.

Minha Jornada (Timeline)

https://app / aluno/timeline

Sua história como corredor — todos os treinos marcantes, provas, recordes pessoais e conquistas em ordem cronológica.

  • O que aparece: treinos de destaque, provas completadas, recordes pessoais (PRs), conquistas desbloqueadas e avaliações

💡 Dica

A timeline é ótima para ver sua evolução ao longo do tempo. Nos momentos de desmotivação, veja o quanto você já progrediu desde que começou!

6. Perguntas Frequentes

Todos os perfis
Esqueci minha senha, como recupero?
Na tela de login, clique em "Recuperar senha" → informe seu e-mail → você receberá um link de redefinição em até 5 minutos. Verifique a caixa de spam se não encontrar. O link expira em 30 minutos.
Posso usar no celular?
Sim! O sistema funciona em qualquer navegador mobile. Para melhor experiência, adicione à tela inicial seguindo as instruções da seção 2.3 — ele funciona como um aplicativo nativo.
Meu treinador não vê meus treinos. O que aconteceu?
Confirme que você está vinculado ao treinador correto. Peça para seu treinador verificar na seção Gerenciar Atletas se seu nome aparece na lista. Se não, solicite que ele te adicione novamente.
Como cancelar minha assinatura?
Acesse Perfil → Assinatura → Cancelar assinatura. O cancelamento não é imediato — você continua com acesso completo até o fim do período já pago. Não há reembolso proporcional.
Os dados ficam salvos se eu remover o atalho do celular?
Sim! Todos os dados ficam armazenados no servidor, não no seu celular. Remover o atalho da tela inicial é como fechar um site — seus dados continuam lá. É só adicionar o atalho novamente.
Posso ter mais de um treinador?
Por padrão, cada atleta é vinculado a um treinador. Para casos especiais com múltiplos treinadores, entre em contato com o administrador da sua assessoria para verificar a configuração do plano contratado.
A IA Treinadora guarda o histórico das conversas?
O histórico fica disponível durante a sessão ativa. Ao fechar o navegador ou sair do app e abri-lo novamente, uma nova conversa começa do zero. Isso protege sua privacidade.
Como exportar meus dados de treinamento?
Treinadores podem exportar dados em CSV ou PDF via a seção Relatórios. Atletas podem solicitar a exportação dos próprios dados ao treinador ou enviar e-mail para o suporte com o pedido formal.
O sistema funciona sem internet?
O PACE RUN PRO é uma aplicação web e requer conexão para carregar dados. Porém, uma vez carregada a página do treino de força, o timer funciona mesmo com instabilidade de rede. Para melhores resultados, tenha conexão ativa.
Como alterar meu e-mail ou foto de perfil?
Clique no ícone de usuário no canto superior direito → Meu perfil → Editar. Para e-mail, uma confirmação será enviada para o novo endereço antes da alteração ser concluída.
