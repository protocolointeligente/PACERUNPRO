# Objetivo

Definir a evolução da ENKY desde a primeira versão pública até a consolidação como a principal plataforma científica de treinamento esportivo.

Este roadmap não é baseado em datas.

É baseado em maturidade do produto.

Cada release só será iniciada quando a anterior atingir os critérios mínimos de qualidade.

A qualidade sempre prevalece sobre a velocidade.

---

# Filosofia

Não construiremos dezenas de funcionalidades superficiais.

Construiremos poucos módulos extremamente sólidos.

Cada release deverá entregar valor completo.

Nunca funcionalidades pela metade.
